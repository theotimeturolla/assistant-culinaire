




INSTRUCTIONS POUR L'ASSISTANT CULINAIRE

DEUX MÉTHODES POSSIBLES :

MÉTHODE SIMPLE MAIS CA M'ARRANGE PAS (UTILISE MA CLÉ API) :
- Double-cliquez simplement sur AssistantCulinaire.bat 
- Attention : cette méthode utilise ma clé API personnelle qui a un nombre d'utilisations limitées : donc la méthode ci dessous est mieux.

MÉTHODE RECOMMANDÉE (AVEC VOTRE PROPRE CLÉ API) :

0) Créez un dossier sur votre ordinateur pour y mettre tous les fichiers (interface2.py et CodeAssistCuli.py dispos sur le GitHub dans ma branche et dans le dossier que je vous ai envoyé)
1)Installez Visual Studio Code si vous ne l'avez pas déjà
2) Obtenez votre propre clé API Google Gemini (si ce n'est pas déja fait) :
   - Rendez-vous sur https://makersuite.google.com/app/apikey
   - Créez un compte et générez une clé API
3) Créez un fichier nommé exactement ".env" (sans extension c'est important, pas de .txt etc : possible depuis VS Code) contenant:
   GOOGLE_API_KEY=votre_clé_ici 
et placez le dans le dossier crée précédemment à l'étape 0
4) Ouvrez VS Code et ouvrez les fichiers interface2.py et CodeAssistCuli.py
5) Dans le terminal de VS Code, exécutez ces commandes:
   - cd [chemin vers votre dossier] (Le dossier créé à l'étape 0)
   - pip install google-generativeai
   - python interface2.py
6) Amusez vous et dégustez !!
